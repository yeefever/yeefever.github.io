# Blog Posts

Add new posts by creating markdown files with the format:

```
YYYY-MM-DD-slug.md
```

Example: `2025-02-15-on-learning.md`

## Required frontmatter

```yaml
---
title: Your Post Title
date: 2025-02-15
blurb: Optional short summary shown on the blog index
---
```

The rest of the file is standard markdown.
